# 🎓 Graduate Admission Prediction using Machine Learning

## 📌 Overview
This project predicts the probability of a student's admission into graduate programs based on academic performance and profile attributes using Machine Learning techniques.

Universities evaluate applicants using multiple parameters such as GRE score, TOEFL score, CGPA, research experience, and recommendation strength. This project applies a supervised machine learning regression model to estimate the **Chance of Admission**.

---

## 📂 Dataset Information
The dataset contains **500 student records** with academic and profile details.

### Features Used
- **GRE Score** – Graduate Record Examination score
- **TOEFL Score** – English proficiency test score
- **University Rating** – Rating of university (1–5)
- **SOP** – Statement of Purpose strength
- **LOR** – Letter of Recommendation strength
- **CGPA** – Undergraduate GPA
- **Research** – Research experience (0 = No, 1 = Yes)
- **Chance of Admit** – Probability of admission *(Target Variable)*
